## 1. 폴더 구조 (필수)

레포지토리 최상단에 **주차별 폴더**에 **본인 이름** 폴더를 두어 파일을 관리합니다.

```text
repo-root/
 └─ Week1/
     ├─ 홍길동/
     │   ├─ 사진1.png
     │   └─ 사진2.png
     ├─ 허광민/
     └─ 류태현/
```

---

## 2. Pull Request (PR) 생성 방법

| 단계 | 내용 |
|------|------|
| **1. Fork** | 원본 레포지토리를 본인 계정으로 Fork합니다. |
| **2. Clone** | Fork한 레포지토리를 로컬에 Clone합니다. |
| **3. Commit & Push** | 과제 제출 파일을 폴더 구조에 맞춰 추가한 뒤, 본인 원격 레포지토리에 Push합니다. |
| **4. PR 생성** | 본인 레포지토리의 **Pull requests** 탭에서 원본 레포지토리의 `main` 브랜치로 PR을 보냅니다.|
| **5. PR 제목** | `[Week N] 이름 과제 제출` 형식을 사용합니다. (예: `[Week1] 허광민 과제 제출`) |
